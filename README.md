# Logging Spring Boot Starter

Spring Boot starter для автоматического логирования времени выполнения методов в слоях `@RestController` и `@Service`.

Стартер перехватывает вызовы методов через AOP и записывает время выполнения в лог. Подключается как обычная зависимость и не требует дополнительной настройки.

---

## Features

- автоматическое логирование времени выполнения методов
- поддержка методов классов, аннотированных `@RestController` и `@Service`
- реализация через AOP
- простое подключение как зависимости
- не требует дополнительной конфигурации

---

## Tech Stack

- Java
- Spring Boot
- Spring AOP

---

## How It Works

Стартер использует **Aspect-Oriented Programming (AOP)** для перехвата вызовов методов.

Аспект:
- перехватывает методы классов, помеченных `@RestController` и `@Service`
- измеряет время выполнения метода
- записывает результат в лог
